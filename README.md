Fan site and SEO experiment for Steven Spielberg's upcoming UFO movie **Disclosure Day** (June 12, 2026).

**https://disclosureday.nicedreamzwholesale.com**

## What's Here

- **40+ SEO pages** — Programmatic SEO targeting long-tail keywords
- **AI Chat** — Mysterious "D.I.S.C.O." chatbot powered by GPT-4
- **Cast/Crew profiles** — Emily Blunt, Josh O'Connor, Steven Spielberg, John Williams, etc.
- **Topic pages** — Release date, trailer, plot, IMAX, streaming, etc.
- **Comparison pages** — vs Close Encounters, E.T., War of the Worlds, Arrival
- **Theory pages** — Fan speculation about alien intentions, government cover-ups

## Tech Stack

- Static HTML/CSS (no framework, fast loading)
- PHP backend for AI chat
- OpenAI GPT-4o-mini for chatbot
- Nginx on Ubuntu VPS
- JSON-LD structured data for SEO

## Setup

1. Clone repo
2. Copy `chat-api.example.php` to `chat-api.php`
3. Add your OpenAI API key
4. Deploy to any web server
